XHR Monitor Configuration Test

This page is for testing the configuration window.

Test Instructions:

  1. Install the userscript in your browser
  2. Navigate to: https://github.com/JSREI/js-xhr-hook?configuration=true
  3. The configuration window should automatically open
  4. Or use the userscript menu to open configuration

Expected Features:

  • Beautiful modern UI with gradient background
  • Toggle switches for enable/disable options
  • Input fields for configuration values
  • Save, Reset, and Close buttons
  • Configuration persistence using GM_setValue/GM_getValue
  • Success/error notifications

Configuration Options:

  • Monitor Settings: Enable monitoring, logging, breakpoints
  • Filter Settings: URL patterns, HTTP methods
  • Display Settings: Max requests, timestamps
